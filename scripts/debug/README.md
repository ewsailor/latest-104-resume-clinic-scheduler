# 除錯腳本目錄

這個目錄包含各種臨時的除錯和測試腳本，用於解決特定的問題。

## JavaScript 除錯腳本

### `test_api_fix.js`

- **用途**：測試 API 修復功能
- **功能**：發送測試時段資料到 API 並驗證回應
- **使用方式**：在瀏覽器控制台中執行

### `test_chat_session_debug.js`

- **用途**：除錯聊天會話功能
- **功能**：檢查聊天狀態和會話管理
- **使用方式**：在瀏覽器控制台中執行

### `test_giver_id_debug.js`

- **用途**：除錯 Giver ID 設定問題
- **功能**：檢查靜態和動態 Giver 卡片的 ID 設定
- **使用方式**：在瀏覽器控制台中執行

### `test_giver_id_fix.js`

- **用途**：修復 Giver ID 相關問題
- **功能**：測試 Giver ID 修復功能
- **使用方式**：在瀏覽器控制台中執行

### `test_wang_shi_yi_fix.js`

- **用途**：修復特定使用者（王拾壹）的問題
- **功能**：測試特定使用者的功能修復
- **使用方式**：在瀏覽器控制台中執行

## Python 測試腳本

### `test_wang_shi_yi_python.py`

- **用途**：測試特定使用者（王拾壹）的時段提交功能
- **功能**：發送 API 請求測試時段建立
- **使用方式**：`python scripts/debug/test_wang_shi_yi_python.py`

## 注意事項

1. **臨時性質**：這些腳本都是為了解決特定問題而建立的，可能很快就過時
2. **除錯用途**：主要用於開發和除錯，不建議在生產環境中使用
3. **定期清理**：建議定期檢查並刪除不再需要的腳本
4. **使用前檢查**：使用前請確認腳本內容是否符合當前需求

## 使用建議

1. **開發階段**：在開發新功能時，可以參考這些腳本的除錯方法
2. **問題排查**：遇到類似問題時，可以參考對應的除錯腳本
3. **學習參考**：這些腳本展示了不同的除錯技巧和測試方法
